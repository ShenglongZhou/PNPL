# PSNP

This Matlab solver was created based on the algorithm proposed by

S. Zhou, C. Xiu, Y Wang, and D Peng, Revisiting Lq (0 <= q < 1) Norm Regularized Optimization, [arXiv:2306.14394](https://arxiv.org/abs/2306.14394), 2023.

Please give credit to this paper if you use the code for your research.
